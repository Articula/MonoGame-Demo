# MonoGameDemo

A little platformer fun with C# and MonoGame. Mostly random generated level with different kinds of walls, some collectables and a roaming enemy.


## Default Controls

### Keyboard

Left: [Left]  
Right: [Right]  
Jump: [Up]  
Pause: [Enter]  
Reset Level: [F5]  
Exit: [Esc]


### GamePad

Left: [Left (DPad)]  
Right: [Right (DPad)]  
Jump: [A]  
Pause: [Start]  
Reset Level: [Left Shoulder/Bumper]
